# 우주 슈팅 게임

간단한 2D 우주 슈팅 게임입니다.

## 설치 방법

1. Python이 설치되어 있어야 합니다.
2. 필요한 패키지를 설치합니다:
   ```
   pip install -r requirements.txt
   ```

## 게임 실행 방법

다음 명령어로 게임을 실행합니다:
```
python shooting_game.py
```

## 게임 조작 방법

- 왼쪽 화살표 키: 왼쪽으로 이동
- 오른쪽 화살표 키: 오른쪽으로 이동
- 스페이스바: 총알 발사

## 게임 규칙

1. 파란색 사각형(플레이어)을 조작하여 빨간색 사각형(적)을 피하세요.
2. 스페이스바를 눌러 총알을 발사하여 적을 처치하세요.
3. 적을 처치할 때마다 점수가 1점씩 올라갑니다.
4. 적과 충돌하면 게임이 종료됩니다.
